# HackRush-PS-3-A
The Archivist Puzzle
Team: Sherlock
Event: Hackrush 2026 - Machine Learning Problem Statement
Project Overview
This repository contains a sophisticated NLP and Graph Theory pipeline designed to solve a modern version of "Cain’s Jawbone". Our goal was to reconstruct the original sequence of two shuffled mystery novels (Book A and Book B) by analyzing narrative flow, linguistic continuity, and chapter-based constraints.
